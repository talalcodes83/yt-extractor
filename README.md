# YT Extractor

A simple desktop YouTube downloader built with Python, yt-dlp, and Tkinter.

Features
- High quality video extraction
- Audio extraction (mp3/m4a)
- Live progress logs
- Cancel downloads
- GUI interface

Requirements
- Windows
- FFmpeg installed

Download

Download the latest executable from the Releases section.

Build from source

pip install -r requirements.txt
python main.py
